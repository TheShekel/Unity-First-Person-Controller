Controls:

WASD - movement

C - Crouch

Left-Shift - Sprint

Spacebar - Jump

Right-Click - Zoom

E - Interact

Information:

Red box will decrease HP.

Green box will heal HP.

simply walk up to the purple pillar to climb it.

Walk up and look at the white floating cube to start an "interaction."
